Use the free open source app

The Curator app provides a UI, API, virtual office and an automated voting bot with extreme customization for your curation needs. Its MIT license means support for branding with themes (coming soon).

Join the curation network

With APIs pool from most of Curator apps, we provide a very powerful API to access and aggregate curation data from all connecting curator nodes.

About us

Steem Curation Center provides software and networking support for curating content on the Steem Blockchain.

Initially developed at Peer Query with the support of Utopian-io and the Steem community, the project is now maintained by the Steem Institute.
